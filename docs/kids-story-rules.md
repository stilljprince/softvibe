# Kids Story Preset Specification

Target age: 4–9 years

Tone:
- Gentle
- Warm
- Safe
- Imaginative but grounded

Language:
- Short sentences
- Simple vocabulary
- No complex metaphors
- No sarcasm
- No irony

Themes allowed:
- Friendship
- Curiosity
- Nature
- Small magical elements
- Family warmth
- Bedtime calmness

Themes forbidden:
- Fear-based tension
- Monsters as threat
- Death
- Existential dread
- Aggressive conflict

Structure template:

Intro → Setting → Small adventure → Emotional moment → Calm ending → Sleep cue

Ending must always:
- Slow pacing
- Reduce stimulation
- Encourage relaxation
- Fade gently

Prompt builder must apply these rules automatically unless the user explicitly overrides.
