# Prompt Chips – SoftVibe

Prompt Chips assist users in generating structured, high-quality prompts.

Each genre contains categorized chip groups.

Genres:

- ASMR
- Sleep Story
- Meditation
- Kids Story

Chips must:
- Be short
- Insert clean natural language
- Never inject unsafe content
- Never override safety rules

Chips are helpers.
Users can edit final prompt before generation.

Prompt Assistant Flow:
1. User writes prompt
2. Clicks “Improve”
3. Backend returns optimized prompt
4. User sees and edits optimized version
5. Final version is used for script generation
